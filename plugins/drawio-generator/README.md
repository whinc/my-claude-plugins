# Draw.io Generator

生成草图风格的 Draw.io 流程图文件

## 功能特点

- **草图风格**：手绘风格的流程图，线条粗犷自然
- **Google 配色**：使用 Google 经典四色方案
- **多种形状**：支持角色、命令、处理块、文档等多种形状
- **自动命名**：基于描述自动生成文件名

## 安装

```bash
# 复制插件到 .claude-plugins 目录
cp -r drawio-generator ~/.claude-plugins/
```

## 使用方法

```bash
# 生成流程图
/gen-drawio 用户登录流程：用户输入密码 -> 系统验证 -> 登录成功

# 生成复杂流程
/gen-drawio 订单处理：接收订单 -> 库存检查 -> 支付 -> 发货
```

## 形状说明

- **椭圆形**：角色/人物
- **平行四边形**：斜杠命令
- **矩形**：处理块/Skill
- **文档形**：文档/文件
