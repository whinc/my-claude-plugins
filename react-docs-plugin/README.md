# React Docs Plugin

A React documentation and utilities plugin for Claude Code. This plugin provides specialized React documentation capabilities through skills.

## Description

The react-docs-plugin is currently in its initial state and provides a basic structure for future React documentation functionality.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add ./.claude-plugin/marketplace.json
```

Then install the plugin:

```bash
/plugin install react-docs-plugin
```

## Skills

### `ahooks`

A React hooks library documentation skill that provides specialized knowledge and examples for the ahooks library.

## Current Status

This plugin is currently empty and serves as a foundation for future React documentation functionality.
