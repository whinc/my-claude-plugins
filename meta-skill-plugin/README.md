# Meta Skill Plugin

A meta-plugin for creating skills. This plugin serves as a framework for skill creation functionality.

## Description

The meta-skill-plugin is currently in its initial empty state and provides a basic structure for future skill creation capabilities.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add ./.claude-plugin/marketplace.json
```

Then install the plugin:

```bash
/plugin install meta-skill-plugin
```

## Commands

### `/skill-creator`

A basic placeholder command that demonstrates the plugin structure.

## Current Status

This plugin is currently empty and serves as a foundation for future skill creation functionality.
